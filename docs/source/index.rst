Welcome to scrapista's documentation!
=====================================

Welcome to the complete beginner's guide to Scrapista! Scrapista is a tool that helps you scraping useful information from some of the most well known websites such as Amazon, Wikipedia or IMDb. 

Who is this package for?

This package is for everyone who don't know much about scraping information from websites or for the ones who know how to scrape but looking for an easier and faster way of doing it. 

If you are completely new to scrapista, :ref:`install scrapista first <settingup>` then get started with it. 

If you already have scrapista installed on your system, you can :ref:`check if everything is working well <importingscrapista>` or skip the set up phase and :ref:`get started with scrapista <gettingstarted>`.
